Kinoko Talk 利用規約

1. 本サービスの概要
Kinoko Talk（以下「本アプリ」）は、スマートフォンのカメラ映像をAIが解析し、キノコとのエンターテインメント会話体験を提供するWebアプリケーションです。

2. 免責事項
本アプリが提供するキノコの種類判別や状態分析は、AIによるエンターテインメント目的の推測であり、正確性を保証しません。キノコの食用可否の判断には絶対に使用しないでください。野生キノコの採取・摂食については必ず専門家にご相談ください。誤った判断による健康被害等について、開発者は一切の責任を負いません。

3. 利用条件
APIの利用料金はユーザーまたはサービス提供者の負担となります。APIキーをご自身で入力した場合は、ブラウザ内（localStorage）にのみ保存され、開発者のサーバーに送信されることはありません。

4. 禁止事項
違法な目的での使用やAPIの不正利用を禁止します。

5. 変更
本規約は予告なく変更される場合があります。
Kinoko Talk プライバシーポリシー

1. 収集する情報
・カメラ映像: AI解析のために一時的に使用し、保存されません。
・音声データ: ブラウザ内で処理され、保存されません。
・APIキー: ブラウザのlocalStorageにのみ保存されます。

2. データの送信先
画像と音声テキストはGoogle (Gemini API) またはCloudflare Workers プロキシ経由で送信されます。各サービスのプライバシーポリシーをご確認ください。

3. データの保存
会話履歴はブラウザのメモリに一時保持され、ページを閉じると消去されます。独自のサーバーにデータは保存されません。

4. Cookie
本アプリはCookieを使用しません。

5. お問い合わせ
本ポリシーに関するお問い合わせはアプリ開発者までご連絡ください。
